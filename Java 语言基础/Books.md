## Java 语言基础的相关书籍推荐

==**Thinking in Java**==

这本书不用多做解释，人手一本，字典一样的去使用它。基本语法、关键字和 IO 系统的学习基本就仰仗它了。豆瓣评分：8.9

==Java 多线程核心技术==

多线程并发入门书籍，虽然很多评论说书中例子太多，描述啰嗦，但是那是对于他们而言，对于一个未入门的人来说，那是极好的。豆瓣评分：5.3

==Java 并发编程实战==

并发的进阶类书籍，豆瓣 9.0 的评分。

==Java 8 实战==

java 8 算是 Java 与时俱进的一个标志，与此同时也诞生了很多新特性，包括Lambdas、流和函数式编程等。这些新特性以及 jdk 对于某些集合的重新实现或者修改实现都非常值得学习。豆瓣评分：9.1

==深入理解 Java 虚拟机==

这本书已经成为国内学习 jvm 的一个样本书了，是学习 jvm 的一个入门书籍。豆瓣评分：8.9
